FAQ

Do I need to be logged in?
If you use the app on multiple devices, make sure you’re using the same profile/session so your referral stats stay consistent.
Can I refer multiple friends?
Yes! There’s no limit. Each friend who completes a quiz earns you CP.
When do I get the CP?
As soon as the milestone quiz is completed. If your leaderboard rank doesn’t update immediately, refresh the page.
